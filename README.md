# GalaxyRend_addon
Blender addon for the GalaxyRend distributed render network

## Installation

1. Download or clone this repository
2. In Blender, go to `Edit > Preferences > Add-ons`
3. Click `Install...` and select the `__init__.py` file from this repository
4. Enable the addon by checking the checkbox next to "GalaxyRend"

## Usage

After installation:

1. Open the sidebar in the 3D Viewport by pressing `N` on your keyboard
2. Find the "GalaxyRend" tab in the sidebar
3. Use the **"Create GalaxyRend Workspace"** button to create a dedicated workspace
4. Enter text in the text field
5. Click **"Submit"** to submit your input

## Features

- **Custom Workspace**: Creates a dedicated GalaxyRend workspace tab in Blender
- **Side Panel**: A panel in the 3D View sidebar with:
  - A text input field
  - A submit button

## Requirements

- Blender 2.80 or later
